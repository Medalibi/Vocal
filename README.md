The purpose of the application of sending mail by voice recognition is to assist
 disabled people wrote and send e-mails through an interactive interface voice
 recognition and voice or sound response to each task performed by the program.

This application is a Java Enterprise Edition based application developed at
 the Faculty of Science of Tunis by a group of student from the Professional
 Master of Network System and Telecommunication, as a Mini-Project supervised
 by Mr. Riadh Bouhouch. It’s based on Java Speech API for the voice recognition.
